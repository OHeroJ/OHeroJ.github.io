Runner

Runner 可以分布在不同的主机上，同一个主机上也可以有多个 Runner.
gitlab-runner 可以分为两种类型：

  • Shared Runner : 系统管理员创建
  • Specific Runner: 有该工程访问权限的人可以创建

每次 gitlab-ci-multi-runner 注册完成之后，还必须执行 run 操作，否则它无法使用
