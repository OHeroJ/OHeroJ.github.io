2016 -> 2017

发表于 2017-01-01 | | 阅读次数:

随着滴答滴答的尾声，2017年的第二天也即将到来。一天的逝去，无不叹伤时间流水般的飞快。

16年我干了什么，17年我该干什么？ 每年开端都会有这么个疑问，
但是都找不到明确的答案，我只能说我是个失败者！

16年接触很多东西，也实践很多东西，貌似有点小成，但心里总发出个声音做的不够彻底，还留着很多 how 和 why, 不得不承认我内心还是发虚的。

16年主要干了这么几大留存在大脑的事情：

  • 公司破产，转战深圳，工作难找
  • 成为公司基础研发者，实时组件化工作
  • 学习 Go、React、Python、Shell、Git

大体情况可全圈可点，代码质量有所提高、都未深入消化做到尽善尽美，唯一的就是自己可以组件看清自己的不足，可以找到发力点，这是我觉得这一年我最大的收获。

17年，目标扎实基础，尽善尽美，把那些无意识的代码都弄清楚, 建立自己的技能点, 把之前的所有技能树都进行强化和融汇体现在某些开源项目中，巩固更多的工具的使用。有这么几点：

  • 设计模式, UML 画图
  • 强化个语言树(按月)
  • 熟练基本算法实现
  • 项目化（流程、自动化、构建，git hook…，单元测试UI测试）
  • 融合知识点
  • 读开源 nx 代码库（每周一个且写博客总结收获点）
  • iOS8,9,10 相关特性的使用
  • 博客质量提高， 搭建个人站点，写一个自己的 App 且上线长期维护
  • 锻炼身体，将体重降到 130，去除小肚子

17年是一个充实自己的阶段，挺住我就牛逼，没挺住，那我就还是16年的我！

Believe in yourself， i think i can do.

要么继续崇拜别人，要么让别人崇拜我!!!

# 总结
Swift 之 Vapor小试
