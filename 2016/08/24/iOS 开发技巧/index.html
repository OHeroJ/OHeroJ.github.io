XCode快捷键

  1. cmd + shift + j : 定位当前代码文件所在的目录中的位置
  2. cmd + shift + o : 寻找类对应的代码文件
  3. cmd + shift + f : 从全局搜索全局文件
  4. cmd + \ : 取消断点 或者打断点
  5. cmd + alt + <- : 折叠代码
  6. cmd + alt + -> : 展开代码
