发表于 2018-01-03 | | 阅读次数:

创建

Haskell 是如何定义 Tree：

二叉树： 该节点有两个子节点， 这个子节点要么也是二叉树，要么是空节点。

swift 定义:

红黑二叉树

红黑树, 是最常用的持久数据结构。

1）每个结点要么是红的，要么是黑的。
2）根结点是黑的。
3）每个叶结点（叶结点即指树尾端NIL指针或NULL结点）是黑的。
4）如果一个结点是红的，那么它的俩个儿子都是黑的。
5）对于任一结点而言，其到叶结点树尾端NIL指针的每一条路径都包含相同数目的黑结点。

树的旋转

左旋

当某个节点 pivot 上，做左旋操作时， 假设它的右孩子 Y 不是 Nil， pivot 可以为任何不是 nil 的左孩子节点。

左旋以 pivot 到 Y之间的链为 支轴, 它使 Y 成为该孩子树新的根， 而 Y 的左孩子 b 则为 pivot 的右孩子。

以某一个 pivot 为指点进行左旋，其变为其右边孩子的左孩子，其右孩子的左孩子变为为 pivot 的右孩子

右旋

pivot 为其左孩子的右孩子， 其左孩子为他右孩子的的右孩子
